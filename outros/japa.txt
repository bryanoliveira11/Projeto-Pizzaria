O site se inicia com uma tela listando todas as pizzas (cardapio), com uma breve descricao e preço. Deve mostrar um máximo de 6 pizzas por pagina e,ao clicar na pizza, deve se exibir uma outra pagina contendo somente a pizza clicada, seu preço e seus ingredientes. O cliente poderá escolher o tamanho da pizza. Pequena, media ou grande. tamanho altera o preço.

Deve conter uma navbar com uma mensagem de boas-vindas, com um link para o cardapio, um link para acessar o formulário do cadastro de clientes, um link para acessar a tela de login e um botão para fazer logout. A navbar tambem deve conter uma caixa de pesquisa, para trazer as pizzas ou ingredientes / sabores pesquisados. Exibir mensagem caso a pesquisa não traga resultados. A navebar terá um carrinho de compras que abre uma tela listando os itens adicionados no carrinho, deve ser possível remover do carrinho também. Caso o Carrinho esteja vázio, o ícone não aparecerá na navbar e, caso o usuário acesse via url, mostrará uma mensagem informando que está vázio.

No cadastro dos clientes devem estar presentes o nome, sobrenome, cpf, cep, rua, bairro, cidade, estado  Os clientes poderão fazer o login na aba login. Nenhum dos campos podem estar vazios, o campo de cep precisa ter 8 digitos não necessariamente válidos para um cep real. O cpf precisa bater de fato, isto é, numeros aleatórios não funcionarão, mas um cpf real sim.
