<h1> Project Pizzeria </h1>

WebSite That Simulates a Pizzeria Ordering.

Project Made Using : 

* Python
* Django Framework
* HTML
* CSS
* BOOTSTRAP
* JAVA-SCRIPT
* SQLITE as DataBase

This WebSite Has :

* Functional Cart Shop System
* Functional Search Bar
* User Register
* Login / Logout
